Nodemon

  • a tool that helps develop node.js based applications by automatically restarting the node application when file changes in the directory are detected.
  • npm install -g nodemon - inslall globally to your system path
  • 'nodemon app.js' - enter mode
  • 'ctrl + C' x2 - exit mode
  • 'npm i -D nodemon' - install nodemon as a development dependency
